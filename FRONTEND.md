# Frontend Development Documentation

## Текущий статус
Реализована верстка основных страниц блога (список и статья) в соответствии с макетами. Все страницы используют единую сетку и обновленный дизайн подвала и шапки.

## Технологии
- **Framework:** Laravel 12
- **CSS:** Tailwind CSS v4 (CSS-first configuration)
- **Fonts:** Google Sans Flex (Локальный вариативный шрифт)
- **Icons/Logos:** SVG

## Дизайн-система и Сетка
- **Основной контейнер:** `max-w-5xl` (центрированный). Все элементы (Header, Content, Footer) выровнены по этой ширине.
- **Шрифт:** `Google Sans Flex` установлен как основной (`font-sans`).
- **Цвета:**
    - Background: `#ffffff`
    - Footer Background: `#323232`
    - Accent Green: `#22c55e`
    - Accent Cyan: `#6dcde3`

## Реализованные компоненты

### 1. Layout (`layouts/site.blade.php`)
- **Header:** Логотип слева, навигация по центру, кнопка "Anfrage senden" справа. Всё выровнено по краям `max-w-5xl`.
- **Footer:** Выполнен в виде "карточки" с закруглением `1.5rem`. 
    - Логотип: Серая иконка + белый текст (`logo-footer.svg`).
    - Кнопка: С белой окантовкой, расположена под навигацией справа.
    - Copyright и ссылки (Impressum, Datenschutz) в нижней плашке.

### 2. Список статей (`blog/index.blade.php`)
- Сетка из 3-х колонок.
- Карточки со скруглением, тенью и эффектом наведения на изображение.
- Категории в виде круглых бейджей.
- Кнопка "Mehr anzeigen" для подгрузки статей.

### 3. Страница статьи (`blog/show.blade.php`)
- Хлебные крошки ("< Blog").
- Заголовок H1 (крупный, Montserrat/Google Sans).
- Hero-изображение со скруглением.
- Типографика контента настроена через Tailwind Prose в `app.css`.

## Ассеты (Assets)

### Шрифты (`public/fonts/`)
- `GoogleSansFlex-Variable.ttf` — основной шрифт проекта.

### Изображения (`public/images/`)
- `logo.svg` — цветной логотип для шапки.
- `logo-white.svg` — полностью белый логотип.
- `logo-gray.svg` — полностью серый логотип.
- `logo-footer.svg` — специальная версия для футера (серая иконка + белый текст).

## Инструкции для разработчика

### Сборка фронтенда
При внесении изменений в Blade-шаблоны или CSS:
```bash
npm run build
```

### Запуск сервера
```bash
php artisan serve
```

### Добавление контента
Контент статьи рендерится через `{!! $post->content !!}`. Стили для заголовков, списков и ссылок внутри контента описаны в `resources/css/app.css` в секции `.prose`.
